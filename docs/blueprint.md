# **App Name**: DigiCard Studio

## Core Features:

- Visual Editor: Drag-and-drop interface for creating digital business cards with real-time preview.
- Template Library: Selection of professionally designed templates to start with, filterable by category.
- Profile Information Input: Entry of personal and professional information, including name, job title, bio, and contact details.
- QR Code Generation: Automatic generation of a QR code dynamically updated with your card information for easy sharing.
- Image Upload: Upload profile pictures or brand logos to customize the digital card's appearance.
- Printing Support: Assistance tool that adds bleed lines and cut-guides.
- AI Template Suggestion: Suggest a suitable template and associated theme options using the provided details. The reasoning is provided as a tool.

## Style Guidelines:

- Primary color: Deep blue (#4263EB), conveying professionalism and trust. 
- Background color: Light gray (#F9FAFB), providing a clean and modern canvas that doesn't distract.
- Accent color: Teal (#2DD4CF), used sparingly for interactive elements and highlights.
- Headline font: 'Space Grotesk' for headings, a modern sans-serif to convey innovation.
- Body font: 'Inter' for body text, to maintain readability and a clean aesthetic.
- Use Material Symbols Outlined for a consistent, modern, and recognizable look.
- Mobile-first approach with a clean, single-column layout for easy navigation and viewing.
- Subtle animations for interactive elements (e.g., button hovers) to provide feedback without being distracting.
